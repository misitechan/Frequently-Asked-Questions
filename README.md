# FAQ

## 什么是 QuantumultX？

<details>
<summary>点击查看答案</summary>

QuantumultX 是一个强大的代理客户端，支持 Shadowsocks、HTTP、SOCKS5 等多种协议。它能够捕获所有的 TCP 流量，并通过 Filter 功能按照您自己的方式进行重定向。
</details>
